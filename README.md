# The-Hungry-Rabbit

The game is to move the rabbit left or right using arrow keys.

The carrots falling at the hand of rabbit is considered a caught.

The bomb falling on the head of the rabbit costs a life. Loosing all five lives ends the game.

The game can be paused anytime between the game. The pausing of the game may not the affect the falling items and resuming the
game can place the rabbit at a random position.

Top five high scores are saved and displayed on selecting high score button.
